# AI‑AGI Campus — Homepage

This is a professional landing homepage scaffold for the AI‑AGI Campus project.

Quick start (PowerShell):

```powershell
cd e:\AI-AGI-Campus\frontend\homepage
npm install
npm run dev
```

Open the URL printed by Vite (usually http://localhost:5173).

The page includes a hero section, feature cards, stats and testimonials. Replace content and connect APIs as needed.
